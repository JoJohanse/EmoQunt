# 需要解压本目录下的stock_data\zh_a.7z.001~006.7z

## 解压后文件结构
```
stock_data/
├── zh_a/
│   └── stocks.csv
└── zh_a.7z.001~006
```

## 文件说明

### stocks.csv
包含A股股票代码和名称信息
**字段说明：**
- `code`: 股票代码（6位数字）
- `name`: 股票名称

**示例数据：**
```
code,name
000001,平安银行
000002,万  科Ａ
000004,*ST国华
000006,深振业Ａ
```

### 单个股票文件夹结构

每个股票代码对应一个文件夹，包含完整的行情数据：
例如：600219/hfq/daily/600219_hfq_daily_20230101_20230131.csv

```
zh_a/
├── {股票代码}/
│   ├── hfq/           # 后复权数据
│   │   ├── daily/
│   │   │   ├── {股票代码}_hfq_daily_{开始日期}_{结束日期}.csv
│   │   │   └── {股票代码}_hfq_daily_factor.csv
│   │   └── minute/
│   │       ├── {股票代码}_hfq_minute.csv
│   │       └── {股票代码}_hfq_minute_factor.csv
│   ├── nfq/           # 不复权数据
│   │   ├── daily/
│   │   │   ├── {股票代码}_nfq_daily_{开始日期}_{结束日期}.csv
│   │   │   └── {股票代码}_nfq_daily_factor.csv
│   │   └── minute/
│   │       ├── {股票代码}_nfq_minute.csv
│   │       └── {股票代码}_nfq_minute_factor.csv
│   └── qfq/           # 前复权数据
│       ├── daily/
│       │   ├── {股票代码}_qfq_daily_{开始日期}_{结束日期}.csv
│       │   └── {股票代码}_qfq_daily_factor.csv
│       └── minute/
│           ├── {股票代码}_qfq_minute.csv
│           └── {股票代码}_qfq_minute_factor.csv
```

#### 日线数据 (daily)
**基础数据字段：**
- `时间`: 交易日期
- `开盘`: 开盘价
- `最高`: 最高价
- `最低`: 最低价
- `收盘`: 收盘价
- `成交量`: 成交股数
- `成交额`: 成交金额
- `流通股数`: 流通股本
- `换手率`: 换手率

**因子数据字段：**
- `时间`: 交易日期
- `开盘`、`最高`、`最低`、`收盘`、`成交量`
- `ma5`: 五日均线
- `ma10`: 十日均线
- `ma20`: 二十日均线
- `rsi`: 相对强弱指标
- `dif`: 差离值
- `dea`: 异同平均数
- `macd_hist`: MACD柱状图
- `k值`: KDJ指标K值
- `atr`: 平均真实价格波幅

#### 分钟数据 (minute)
**基础数据字段：**
- `时间`: 交易时间（精确到分钟）
- `开盘`: 开盘价
- `最高`: 最高价
- `最低`: 最低价
- `收盘`: 收盘价
- `成交量`: 成交股数

#### 复权类型说明
- **hfq (后复权)**: 以当前价格为基准，历史价格进行调整
- **nfq (不复权)**: 原始价格，不做任何调整
- **qfq (前复权)**: 以历史价格为基准，当前价格进行调整

## 使用说明
1. 解压后会生成zh_a目录，包含stocks.csv文件
2. 当前只含有沪深300指数的股票数据
